# QueryAvatar

represents the color configuration of a user's avatar. all fields are hex color strings. by default, they are all set to `#C8C8C8`.

## fields

- head_color: `String`
- torso_color: `String`
- left_arm_color: `String`
- right_arm_color: `String`
- left_leg_color: `String`
- right_leg_color: `String`
